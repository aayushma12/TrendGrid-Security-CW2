# TrendGrid

Fashion e-commerce platform — two independently deployable services:

- [`Backend/`](./Backend) — Express + TypeScript + Prisma + PostgreSQL API. See `Backend/README.md` and `Backend/SECURITY.md`.
- [`Frontend/`](./Frontend) — Next.js 16 (App Router) storefront + admin console. See `Frontend/README.md`.

Each subdirectory is set up and run independently — see its own README for setup, scripts, Docker, and CI/CD instructions.
